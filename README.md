# 안녕하세요, 웹 개발자를 준비하고 있는 소채연입니다.

Spring Boot 기반 웹 프로젝트를 중심으로  
사용자에게 필요한 기능을 안정적으로 구현하고,  
화면 흐름과 사용성을 함께 고민하며 개발하고 있습니다.

기능이 정상적으로 동작하는 것뿐만 아니라  
사용자가 서비스를 이용하는 과정에서 불편함이 없도록  
예외 상황, 안내 문구, 화면 구성까지 세심하게 다듬는 개발자가 되고자 합니다.

## 📌 Projects

### 🏕 모닥모닥

캠핑용품 대여와 구매를 함께 제공하는 웹 기반 커머스 프로젝트입니다.

**주요 기능**  
회원가입, 마이페이지, 상품 조회, 장바구니, 주문/결제, 배송조회, 대여, 커뮤니티, 고객센터, 관리자 기능

**담당 역할**

- 로그인/회원가입, 계정 찾기 기능 구현
- 마이페이지 기능 구현
  - 주문 내역, 리뷰 내역, 위시리스트, 최근 본 상품
  - 쿠폰 내역, 포인트 내역
- 멤버십 페이지 구현
- 배송조회 기능 구현
- 통합 검색 기능 구현
- 프로젝트 전체 페이지 CSS 개선 및 UI 스타일 통일 및 부가 기능 추가

### 🧠 AdBye

AI 기반 광고성 리뷰 판별 서비스입니다.

사용자 리뷰를 분석하여 광고성 리뷰와의 유사도, 감정 흐름, 판단 결과를 시각적으로 제공하는 프로젝트입니다.

**주요 기능**

- 리뷰 문장 임베딩
- 광고성 리뷰 유사도 분석
- 감정 분석 결과 제공
- 광고성 여부 판단
- 분석 결과 시각화

**사용 기술**  
Python · SentenceTransformer · all-MiniLM-L6-v2 · SQLite · Matplotlib · React
